# О проекте

Лендинг по макету с моего предыдущего места работы. На работе этот проект был сделан на next.js в интеграции с headless cms strapi. Однако сейчас он по каким-то причинам не открывается. И я решил сделать его заново на другом фреймворке - vue.

## Запуск проекта

- pnpm dev - запуск проекта для разработки
- pnpm build - сборка проекта

## Особенности проекта

- Проект на nuxt
- Для слайдера использован модуль nuxt-swiper
